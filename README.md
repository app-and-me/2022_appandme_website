### 앱앤미 웹사이트

미림여자정보과학고등학교의 전공동아리 앱앤미(AppandMe) 홍보 및 지원 사이트입니다. (2022.03 ~ 운영 중)

#### Website

http://appandme.dothome.co.kr/

### Features

* About
    * 앱앤미 동아리의 간단한 소개를 제공합니다.
* 부원소개
    * 9, 10기 부원들의 소개와 사진을 보여줍니다.
* 지원하기
    * 앱앤미에 지원하려는 신입생들의 온라인 신청을 받고 있습니다. 
* Q&A
    * 동아리에 궁금한 점을 마음껏 물어볼 수 있습니다.
